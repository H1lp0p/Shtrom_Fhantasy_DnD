# Яма страданий

| Свойство | Значение |
|----------|----------|
| Тип | Активная |
| Тип урона | Магический |
| Стоимость ОС | 3 |

| Дистанция | Ближняя (Аура) |
| Цель | На себя (все враги в Ближней дистанции) |
| Длительность | 3 хода |
| Требования | Уровень 5+ |

## Описание

Герой открывает портал в неизведанные глубины, создавая яму темной энергии вокруг себя на 3 хода.

Любой попавший в него враг получает **2d4 + мод. Силы** магического урона и **Обездвиживается**. Обездвиженные враги также получают урон в начале своего хода.
