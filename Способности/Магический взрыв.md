# Магический взрыв

| Свойство | Значение |
|----------|----------|
| Тип | Активная |
| Тип урона | Магический |
| Стоимость ОС | 1 |
| Мана | 10 |
| Действие | Реакция |
| Дистанция | Ближняя (область) |
| Цель | Все враги на Ближней дистанции |
| Длительность | Мгновенно |

## Описание

Герой высвобождает накопленную магическую энергию вокруг себя для защиты.

Когда вам наносится урон, вы можете реакцией нанести **1d8 + мод. Интеллекта** магического урона всем врагам на Ближней дистанции и отбросить их на Среднюю дистанцию.
