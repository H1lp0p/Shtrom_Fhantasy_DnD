# Удар в спину

| Свойство | Значение |
|----------|----------|
| Тип | Пассивная |
| Тип урона | Чистый |
| Стоимость ОС | 2 |

## Описание

Атаки со спины наносят дополнительный урон.

Если герой атакует врага, который не видит его (Невидимость, отвлечен, атакует другого), урон увеличивается на **1d6 чистого урона**.
