# Магические боеприпасы

| Свойство | Значение |
|----------|----------|
| Тип | Пассивная |
| Тип урона | Магический |
| Стоимость ОС | 1 |
| Требования | Дальнобойное оружие или метательное оружие |

## Описание

Вы создаете боеприпасы и снаряды из магии.

Стоимость: **2 MP за выстрел**. Выстрел магическими боеприпасами наносит **1d4 + мод. Ловкости** магического урона.
