# Магический взрыв

| Свойство | Значение |
|----------|----------|
| Тип | Активная |
| Тип урона | Магический |
| Стоимость ОС | 2 |
| Мана | 10 |
| Действие | Реакция |
| Дистанция | Ближняя (область) |
| Цель | Все на Ближней дистанции |
| Длительность | Мгновенно |

## Описание

Когда вам наносится урон, вы можете реакцией нанести **1d4 + мод. Интеллекта** магического урона всем существам на Ближней дистанции кроме себя. За каждую задетую цель вы получаете **Барьер, равный мод. Интеллекта**
