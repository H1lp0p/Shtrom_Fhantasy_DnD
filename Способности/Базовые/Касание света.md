# Касание света

| Свойство | Значение |
|----------|----------|
| Тип | Активная |
| Стоимость ОС | 1 |
| Мана | 5 |
| Действие | Бонусное |
| Дистанция | Средняя |
| Цель | Один союзник ИЛИ один враг |
| Длительность | 1 ход |

## Описание

Наполняет цель божественной энергией, защищая союзников или раня врагов.

Если цель — союзник, она получает **Блок урона = мод. Интеллекта** до конца своего следующего хода. Если цель — враг, он получает **2d4 + мод. интеллекта** магического урона.
