# Круговой удар

| Свойство | Значение |
|----------|----------|
| Тип | Активная |
| Тип урона | Физический |
| Стоимость ОС | 1 |
| Мана | 7 |
| Действие | Стандартное |
| Дистанция | Ближняя |
| Цель | Все враги на Ближней дистанции |
| Длительность | До конца боя (дебафф) |

## Описание

Все враги на Ближней дистанции получают **Урон оружия + мод. Силы** физического урона и получают **Штраф к урону физических атак**, равный **половине мод. Силы**, до конца боя. Эффект не складывается.
