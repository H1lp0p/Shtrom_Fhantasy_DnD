# Уклонение

| Свойство | Значение |
|----------|----------|
| Тип | Пассивная |
| Стоимость ОС | 2 |

## Описание

Герой движется неуловимо, врагам сложнее попасть по нему.

Когда враг в ближней дистанции промахивается по герою, герой может нанести ему **1d6 + мод. Ловкости** физического урона.
