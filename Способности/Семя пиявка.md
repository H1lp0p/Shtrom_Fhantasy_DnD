# Семя пиявка

| Свойство | Значение |
|----------|----------|
| Тип | Активная |
| Тип урона | Магический |
| Стоимость ОС | 1 |
| Мана | 10 |
| Действие | Стандартное |
| Дистанция | Средняя |
| Цель | Один враг |
| Длительность | 2 хода |

## Описание

Сажает в цель истощающее семя. В начале своего хода цель получает урон. Излечивает союзников в ближней дистанции на часть от этого урона.

В начале каждого хода цель получает **1d6 + мод. Интеллекта** магического урона. Союзники в ближней дистанции от цели восстанавливают **50% этого урона** (округляя вниз).
