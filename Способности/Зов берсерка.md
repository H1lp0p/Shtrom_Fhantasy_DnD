# Зов берсерка

| Свойство | Значение |
|----------|----------|
| Тип | Активная |
| Стоимость ОС | 1 |
| Мана | 10 |
| Действие | Стандартное |
| Дистанция | На себя (Область) |
| Цель | Все враги на ближней дистанции |
| Длительность | Мгновенно (провокация), 1 раунд (барьер) |

## Описание

Провоцирует врагов в радиусе Ближней дистанции. Враги, на которых подействовала провокация, в свой следующий ход должны потратить действие на атаку героя, если он находится в зоне досягаемости.

Герой получает **Барьер**, равный значению **Силы** до начала своего следующего хода.
