# Покровительство

| Свойство | Значение |
|----------|----------|
| Тип | Активная |
| Стоимость ОС | 1 |
| Мана | 10 |
| Действие | Бонусное |
| Дистанция | Средняя |
| Цель | Один союзник ИЛИ один враг |
| Длительность | 1 ход |

## Описание

Наполняет цель божественной энергией, защищая союзников или обезоруживая врагов.

Если цель — союзник, она получает **Блок урона = мод. Интеллекта** до конца своего следующего хода. Если цель — враг, она получает **Разоружение** до конца своего следующего хода.
