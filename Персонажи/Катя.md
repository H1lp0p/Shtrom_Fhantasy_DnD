# Атрибуты

| АТР | Значение            | Мод                                |
| --- | ------------------- | ---------------------------------- |
| СИЛ | `INPUT[number:сил]` | `= round((this.сила - 10)/2)`      |
| ЛВК | `INPUT[number:лвк]` | `= round((this.ловкость - 10)/2)`  |
| ИНТ | `INPUT[number:int]` | `= round((this.интеллект - 10)/2)` |

# Способности

```dataview
TABLE WITHOUT ID
  file.link as Способность,
  тип as Тип,
  стоимость_ос as "ОС",
  мана as "MP"
FROM "Способности"
WHERE contains(this.способности_выбранные, file.name)
SORT уровень ASC
```
